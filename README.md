# 📺 Video Service API

Это микросервис для управления видео — создание, просмотр, поиск, обновление, удаление, фильтрация и получение популярных видео. Сервис написан на **Spring Boot** с использованием **JPA**, **Lombok** и **REST API**.

---------------------------------------------------------

## 🚀 Функциональность

- ✅ Создание нового видео
- 🔍 Поиск по названию и/или категории
- 🧭 Лента видео с пагинацией
- 👤 Получение видео по автору
- 📈 Популярные видео
- ♻️ Обновление и удаление видео (только автором)
- 📺 Счётчики просмотров, лайков, комментариев
- 🔒 Защита по автору (через заголовок `X-Author-Id`)

---------------------------------------------------------

## 📂 Структура проекта

- `controller/VideoController.java` – REST API-контроллер
- `entity/Video.java` – модель видео
- `service/VideoService.java` – бизнес-логика
- `DTO/request/VideoRequest.java` – объект запроса (создание/обновление)
- `DTO/response/VideoResponse.java` – объект ответа
- `repository/VideoRepository.java` – доступ к базе данных

---------------------------------------------------------

## 📦 Технологии

---------------------------------------------------------

📬 **API Эндпоинты**
---------------------------------------------------------

🔻 Создание видео
POST /api/videos/create-new-video

---------------------------------------------------------

🔎 Поиск видео
GET /api/videos/search?title=test&category=Music&limit=10&offset=0
---------------------------------------------------------

📺 Лента (feed)
GET /api/videos/feed?limit=10&offset=0

---------------------------------------------------------

👤 Видео по автору
GET /api/videos/by-author/{authorId}

---------------------------------------------------------

🔥 Популярные видео
GET /api/videos/popular?limit=5

---------------------------------------------------------

🆔 Получить видео по ID
GET /api/videos/{id}

---------------------------------------------------------

🧼 Удалить видео
DELETE /api/videos/{id}

---------------------------------------------------------

♻️ Обновить видео
PUT /api/videos/{id}

---------------------------------------------------------


⚠️ **Уникальность**
Поля title, videoUrl, thumbnailUrl — уникальные, поэтому:

нельзя загрузить одно и то же видео повторно

при ошибке будет 500 с сообщением Duplicate entry ...
➜ Измени значения или проверь, есть ли уже такое видео
